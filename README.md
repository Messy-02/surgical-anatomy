## Description
A high-fidelity 3D anatomical model optimized for mechanical analysis and surgical simulation. This repository includes source files and exported STL meshes designed for patient-specific pre-operative planning and biomechanical testing. Disclaimer: This anatomical model is for educational and research purposes only.It is not a medical device and has not been cleared for clinical use in live surgical procedures.

## Technical Details
* **Software:** Blender/ 3D Slicer
* **File formats:**.STL,.OBJ
## Usage
1. Download the STL files from the \'/models/export\' folder.
2. Import into your preferred slicer or Blender software for analysis.
